# Tere tulemast Koodeksi reposse!

Koodeks on lihtne tööriist kümnend-, kahend-, kaheksand- ja kuueteistkümnendkoodis arvude omavaheliseks teisendamiseks. Programmi kasutamine on väga lihtne – lihtsalt loo konto ning hakka teisendama, eelteadmisi pole vaja. Programm salvestab ka viimatise teisenduse.
